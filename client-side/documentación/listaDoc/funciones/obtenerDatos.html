Obtener Datos

  • Explicación Funciones

En la función Obtener Datos, su objetivo es muy simple, como indica el nombre lo que nos devolverá está función serán todos los datos que nos interesan sobre la página web que pensábamos recoger información, a continuación explicaremos su funcionamiento con minuciosidad.

Al principio definimos las variables con las que queremos trabajar y a cada una le asignamos los nombres de los parámetros que esperamos encontrar en el código de la página web.

En cada una de las variables hacemos un “slice” para obtener solamente el fragmento de código con la información que deseamos conseguir y con la que trabajaremos posteriormente.

Acto seguido dentro del “slice” como podemos observar hemos definido un “find” a cada lado donde ejecutaremos el recorte para que nos devuelva la posición de donde se encuentra la información con la que queremos trabajar, de esa forma conseguiremos que la variable contenga la información que necesitamos sin nada adicional.

Al acabar de definir todas las variables, definimos una lista llamada características, que al principio se encuentra vacía.

Creamos un loop que nos añada las características ofrecidas por cada servicio. Para encontrar las características utilizaremos el “slice” y el “find” como recursos igual que los hemos usado anteriormente, pero como en este caso se trata de un array de información queremos ejecutar está orden las veces necesarias hasta que ya no encuentre más características del servicio.

A continuación creamos un diccionario, donde cada campo del diccionario está definido por el nombre identificador de la información´ que esperábamos obtener y cada valor de cada campo está representado con el valor de las variables con las que hemos trabajado.

Finalmente está función nos devolverá toda la información del servicio que deseado en formato de diccionario, con el que nosotros posteriormente añadiremos a nuestra base de datos.

La función obtenerDatos funciona por el principio de SRP (Single Responsibility Principle) ya que esta función tiene una responsabilidad única. En este caso el único objetivo de esta función es buscar los datos o la información que nos interesa de la página web y almacenarla en un diccionario, que más adelante guardaremos en nuestra base de datos de Mongo, aunque está funcionalidad depende de otra función y no de ella.
