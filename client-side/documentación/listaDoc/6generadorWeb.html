Generador de Páginas Web

Este archivo fue creado para poder generar páginas web a gran velocidad. Su intención es poder agilizar el trabajo y aumentar el rendimiento al igual que la productividad dentro del equipo de trabajo.
Básicamente su función es crear páginas web para poder añadir servicios al sitio web de la empresa, ajustándose a los parámetros que ofrecen los distintos servicios ya ofertados y visualizados por los consumidores en la página web de la compañía.

Como podemos ver al inicio, para ejecutar este generador de páginas web, requerimos el uso de la librería os proporcionada por python. Esta librería nos permite acceder a las funcionalidades del sistema operativo, permitiéndonos manipular la estructura de directorios para, por ejemplo, leer y escribir archivos. En este caso utilizaremos la funcionalidad que nos permite escribir archivos proporcionada por esta librería.
El archivo se centra en una única función llamada crearPagina, en la cual le pasamos todas las variables necesarias para poder generar los campos que necesitamos para los servicios de nuestra página web. Declaramos la variable llamada direccion, en la que su funcionalidad es guardar los documentos que generemos con el generados en un directorio específico del proyecto.
La variable file hace referencia al archivo que será generado en la dirección especificada anteriormente y va tener la extensión .html. El documento será abierto en modo w (write) y utilizaremos un encoding utf-8 para que los caracteres que se visualizarán en pantalla tengan un formato legible.
Al haber creado la variable file con la función de os, podemos observar como tiene un nuevo método llamado write, que se encarga de introducir contenido en el archivo. Dependiendo de la gama del servicio a generar, se definirá la tasa de dicho servicio, que es básicamente un pago por el servicio prestado.
Las características, que están contenidas en un array, se introducen una a una gracias a un bucle contador. Una vez se han escrito todas las líneas del código HTML, terminamos la escritura del archivo con el método close.
Una vez definida la función, generamos las primeras 11 páginas web con los servicios que queramos introducir. Estos son contenidos dentro de una variable de tipo array en forma de diccionarios.
