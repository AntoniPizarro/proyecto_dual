Interface

Se trata de una manera muy efectiva y productiva para hacer uso de las diferentes funcones del programa. Su poder sale de la librería tkinter que permite generar ventanas y widgets interactivos.
El código empieza importando una serie de librerías y módulos para poder hacer todas y cada una de las funcionalidades del objetivo de proyecto.
Declaramos la variable ventana donde estará todo el contenido de la interfaz y le configuramos unas dimensiones predeterminadas no limitantes.
El menú está declarado por 3 secciones:

Scrap habilita los widgets necesarios para todo lo relacionado con extraer la información del sitio web. Tiene una opción llamada Obtener todos los datos que activa los widgets que permiten obtener todos los datos de todos los servicios que se encuentren en algún lugar del sitio web.
Generar Producto habilita todos los widgets necesarios para generar un producto con todos los campos que requiera. Contiene 3 opciones principales: Generar en sitio WEB se encarga de utilizar la función principal del Generador de páginas web para generar en el directorio del proyecto especificado, un modelo de página web que requiere una publicación manual al host del sitio web. Esto sirve para prevenir errores humanos. Activa una serie de widgets, que básicamente son 8 campos de entrada, o input fields en los que se especifica el tipo de dato que se debe introducir. También revela las casillas de las características, las cuales solo se permite marcar hasta 6 casillas. Y claro está, el botón que ejecuta la función. Generar en MongoAtlas se encarga de introducir el servicio en la base de datos, con los campos ya explicados. Generar en sitio WEB y MongoAtlas es la combinacion de las dos opciones anteriores.
Ayuda sirve para acceder a toda la ayuda necesaria para entender este proyecto. Tiene una opción llamada Documentación, que al seleccionarla nos lleva a esta misma documentación.

El programa se compone de 7 funciones mas una función comodín, que sirve al desarrollador a comprobar que los widgets que ejecuten una función lo harán correctamente. Cada una de las funciones se encarga de activar y desactivar widgets y utilizar los módulos importados por las librerías.
Todos los widgets están colocados en una cuadrícula que se encarga de alinear los diferentes elementos de la ventana. Cada uno de los widgets es de un tipo diferente y tiene sus propiedades específicas, como lo son el margen y la posición.
